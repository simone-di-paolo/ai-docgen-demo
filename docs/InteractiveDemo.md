# Sezione Demo Interattiva

Questa è una parte fondamentale dell'applicazione, che permette agli utenti di interagire direttamente con l'AI. Consiste in un'interfaccia simile a una chat, arricchita da controlli di personalizzazione.

- **Input:** Un'area di testo in cui gli utenti possono inserire la loro idea di landing page.
- **Button:** Un pulsante "Genera" per inviare l'idea all'AI.
- **Output:** Un'area di visualizzazione in cui appare la risposta dell'AI (ad esempio, codice generato o messaggi di stato).

**Funzionalità di Personalizzazione:**

Questa sezione include ora controlli interattivi che permettono agli utenti di modificare l'aspetto visivo di un componente di anteprima. Le opzioni includono:

- **Colore Tema:** Un selettore di colori che permette di cambiare il colore principale utilizzato nel componente di anteprima.
- **Dimensione Font:** Un input numerico per modificare la dimensione del testo nel componente di anteprima.
- **Tipo di Font:** Un menu a tendina per selezionare il font da utilizzare nel componente di anteprima. Le opzioni disponibili sono: Roboto, Arial, Georgia, Courier New e Brush Script.
- **Modalità Dark/Light:** Un interruttore per attivare o disattivare la modalità scura, influenzando lo schema di colori generale.

Il componente di anteprima visualizza le modifiche in tempo reale. Le proprietà personalizzabili del componente di anteprima includono il colore del titolo, la dimensione del testo e il font utilizzato.
